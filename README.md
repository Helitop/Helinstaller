# 🛠️ Helinstaller
> Твой персональный центр управления Windows. Установка софта, твики и активация в один клик.
> 
> Делал в качестве курсового проекта.
> 
> ![Platform](https://img.shields.io/badge/platform-Windows-0078d4?style=for-the-badge)
>
## ✨ Особенности

- 📦 **Умный инсталлятор** — Поддержка WinGet, GitHub API и прямых ссылок.
- 🎨 **Fluent Design** — Интерфейс на WPF UI, вдохновленный стандартами Windows 11.
- ⚡ **MAS Integration** — Полностью графический интерфейс для Microsoft Activation Scripts (никакой консоли!).
- 🔧 **Твики системы** — Отключение телеметрии, настройка интерфейса и обход блокировок ИИ без VPN.
- 🧩 **Ventoy** — Функционал создания установочного накопителя Windows.
